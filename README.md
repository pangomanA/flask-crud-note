# Flask CRUD Note

A responsive, dark-themed web application for managing personal notes. Developed to master **CRUD** logic, **SQLAlchemy** database persistence, and modern Python workflows.

---

## 🚀 Features

*   **Full CRUD**: Create, Read, Update, and Delete notes.
*   **Database Persistence**: Powered by SQLite and SQLAlchemy.
*   **Modern UI**: Custom-built responsive dark theme.
*   **Fast Workflow**: Managed via `uv` for high-performance dependency handling.

---

## 🛠️ Tech Stack

*   **Backend**: Flask (Python)
*   **Database**: SQLAlchemy / SQLite
*   **Tooling**: [uv](https://github.com/astral-sh/uv)
*   **Frontend**: HTML5, CSS3

---

## 📦 Run the Project

Ensure you have `uv` installed, then run:
```bash
uv run python run.py
